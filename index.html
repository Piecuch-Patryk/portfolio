Technologie, Frameworki i Biblioteki których używam w codziennej pracy

FRONT-END

HTML 5 - używam semantyczne znaczniki do tworzenia layoutów stron.

CSS 3 - potrafię tworzyć responsywne strony i aplikacje internetowe używając media queries. Umiem również tworzyć proste animacje.

JavaScript - tworzę aplikacje frontendowe, które wykorzystują AJAX lub nowe funkcjonalności z ES6 (fetch) do komunikowania się z bazą danych lub z REST API.

jQuery - potrafię manipulować elementami DOM, asynchronicznie podmieniać kontent wraz z animacjami.

Bootstrap - używam do tworzenia responsywnych layoutów stron - mobile first.

Masonry - wykorzystuję do układania zdjęć różnych rozmiarów w siatce.

BACK-END

PHP 7 - potrafię programować zgodnie z OOP. Znam wzorzec projektowy MVC.

MySql - używam relacyjnych i nierelacyjnych baz danych w moich projektach

Laravel - potrafię zaprojektować i napisać aplikację typu CMS zgodnie z zasadami OOP i CRUD.

PHPMailer - ułatwia mi pracę nad wysyłaniem maili z potwierdzeniem/resetowaniem hasła itp.

Intervention - bardzo pomocna biblioteka, która ułatwia manipulowanie zdjęciami.

jenssegers/agent - używam do wykrywania rozdzielczości wyświetlacza, dzięki temu aplikacja ładuje zdjęcia o odpowiednim rozmiarze.

INNE

Narzędzia, Wzorce Projektowe, Języki Obce itp.

Git - używam podstawowe komendy w konsoli do kontrolowania wersji tworzonej aplikacji lub strony.

MVC - umiem oddzielać różne warstwy aplikacji, żeby łatwiej było zarządzać kodem i dodawać nowe funkcjonalności.

OOP - programuje obiektowo w PHP i JS (modules).

Refaktoryzacja kodu - często wracam do wcześniej napisanego kodu i staram się go poprawić.

Gimp - podstawowa umiejętność obróbki grafiki, tworzenie logo, wycinanie elementów itp.

Adobe XD - podstawowa umiejętność tworzenia layoutów stron responsywnych.

Język angielski - komunikatywna znajomość w mowie i piśmie, potrafię korzystać z dokumentacji technicznej.
